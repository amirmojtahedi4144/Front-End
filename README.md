## 👋 درباره من

توسعه‌دهنده فرانت‌اند با تمرکز ویژه بر **رابط‌های کاربری فارسی و راست‌به‌چپ (RTL)**. علاقه‌مند به طراحی رابط‌های تمیز، سریع و بدون وابستگی‌های اضافه، با استفاده از **Tailwind CSS** و **Alpine.js**.

- 🎨 طراحی و پیاده‌سازی رابط‌های کاربری مدرن و پریمیوم
- 🌐 تخصص در پروژه‌های فارسی‌زبان و راست‌به‌چپ
- ⚡ بهینه‌سازی عملکرد و تجربه کاربری
- 🧩 ساختاردهی کد تمیز و قابل نگهداری

## 📂 پروژه‌های منتخب

### 🛍️ آرامیا (Aramia)
پلتفرم مدیتیشن با طراحی کاملاً راست‌به‌چپ، شامل صفحه اصلی و داشبورد برای کاربران، پیاده‌سازی‌شده با Tailwind CSS و Alpine.js.

<div align="center">
<img src="https://via.placeholder.com/800x400/1e293b/d4af37?text=RadMall+Marketplace+Preview" width="80%" alt="پیش‌نمایش پروژه آرامیا" />
</div>

**ویژگی‌های کلیدی:**
- ریل‌های اسکرول پیوسته (Marquee) برای فروشندگان برتر و پیشنهادهای ویژه
- کارت‌های محصول واکنش‌گرا در چهار بریک‌پوینت
- حالت روشن/تاریک با ذخیره‌سازی ترجیحات کاربر
- افکت‌های هاور و درخشش طلایی روی کارت‌های غرفه
